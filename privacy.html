Privacy Policy

V Content Analytics collects the following data through official platform APIs:

  • Public account metrics (follower counts, engagement rates, view counts)
  • Content performance data (likes, comments, shares)

Data Storage

All data is stored as static JSON files in a private GitHub repository. No database or external storage is used.

Data Sharing

No personal data is shared with or sold to third parties.

Contact

For questions, contact the app owner through the dashboard website.
